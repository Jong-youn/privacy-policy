# 4-7-8 호흡 앱 개인정보 처리방침

_최종 업데이트: [2026-07-13]_

본 앱("4-7-8 Breathing", 이하 "앱")은 **[개발자 이름 또는 스튜디오명]**(이하
"당사")이 개발 및 운영합니다. 당사는 이용자의 개인정보를 소중히 여기며,
앱이 다루는 소량의 데이터에 대해 투명하게 안내드리고자 합니다.

## 1. 앱이 수집하지 않는 정보

앱은 **회원가입/로그인 기능이 없으며**, 어떠한 개인식별정보도 요구하지
않습니다. 구체적으로 아래 정보를 수집하지 않습니다.

- 이름, 이메일, 전화번호, 주소
- 위치 정보 (GPS/대략 위치 모두)
- 연락처, 사진, 마이크, 카메라 접근
- 앱 간 추적을 위한 광고 식별자(IDFA)

## 2. 기기 내에 저장되는 정보

앱은 아래 설정을 iOS의 표준 저장소(`UserDefaults`)에 기기 내부에만
저장합니다. 이 데이터는 **기기 밖으로 전송되지 않으며** 앱을 삭제하면
함께 삭제됩니다.

- 표시 언어 설정
- 세션당 호흡 횟수 설정

## 3. 제3자 서비스가 수집하는 정보

앱의 안정적 운영을 위해 Google LLC의 두 가지 서비스("Firebase")를
이용합니다.

### 3.1 Firebase Crashlytics
앱이 비정상 종료되면 Firebase Crashlytics가 자동으로 진단 보고서를
전송합니다. 보고서에는 다음이 포함됩니다.

- 크래시 스택 트레이스
- 앱 버전 및 운영체제 버전
- 기기 모델명 (예: iPhone 13)
- 이용자를 식별할 수 없는 임의의 기기 단위 설치 ID (크래시 영향 기기
  수를 집계하는 용도)

### 3.2 Firebase Analytics
Firebase Analytics는 어떤 기능이 유용하고 어떤 부분을 개선해야 할지
파악하기 위해 익명 사용 통계를 수집합니다. 주요 이벤트는 다음과 같습니다.

- 앱 실행/종료
- 앱 내 화면 조회
- 세션 지속 시간
- 앱 버전, OS 버전, 기기 모델, 언어, 국가 (IP로부터 추정하며 IP 자체는
  저장되지 않음)

**IDFA(광고 식별자)는 사용하지 않습니다.** 앱은 다른 회사의 앱/웹사이트
사용 이력을 추적하지 않습니다.

Google은 위 데이터를 당사를 대신하여 처리하고 Google 인프라에 저장합니다.
Firebase의 데이터 처리 정책은 다음에서 확인하실 수 있습니다:
<https://firebase.google.com/support/privacy>.

## 4. 처리 목적

3항의 데이터는 다음 목적으로만 처리됩니다.

- 크래시 진단 및 안정성 개선
- 이용 현황 집계 분석을 통한 앱 개선
- 서비스 유지 의무 이행

당사는 위 데이터를 광고주나 데이터 브로커에게 판매·대여·공유하지 않습니다.

## 5. 보관 기간

- Firebase Analytics 데이터는 Google의 기본 보관 정책에 따라
  보관됩니다 (현재 이용자 단위 데이터 2개월, 이벤트 데이터 14개월).
- Firebase Crashlytics 크래시 보고서는 Google Firebase 약관이 정한 기간
  동안 보관됩니다.
- 당사는 Firebase 외부에 별도의 데이터 사본을 보관하지 않습니다.

## 6. 이용자의 권리

이용자는 언제든지 다음 방법으로 데이터 전송을 통제할 수 있습니다.

- **분석 데이터 옵트아웃:** 앱을 삭제하시면 해당 기기 설치와 연결된
  Analytics 이벤트 전송이 즉시 중단되며, 설치 ID는 Google 정책에 따라
  삭제 대상이 됩니다.
- **열람 및 삭제 요청:** **[CONTACT_EMAIL]** 로 이메일 주시면 해당 기기
  설치에 연결된 데이터의 열람·삭제 요청을 처리해 드립니다. 대략적인
  일자와 기기 모델을 함께 알려주시면 조회에 도움이 됩니다.

## 7. 아동의 개인정보

앱은 13세 미만 아동을 대상으로 하지 않으며, 당사는 아동의 개인정보를
의도적으로 수집하지 않습니다. 부주의로 그러한 수집이 이루어졌다고 판단되면
연락해 주시면 즉시 삭제하겠습니다.
